### Technologies used:
- [x] Kotlin language
- [x] MVVM (Model - View - ViewModel)
- [x] Coroutines & Flow
- [x] Navigation component (more than one Fragment relies on single Activity)
- [x] RESTAPi, JSON
- [x] Retrofit2, OkHTTP3 (working with out-source Data)
- [x] Timber (for Debuging Code)
- [x] Glide (lib for image loader)
- [x] AwesomeSplash (lib for Awesome Splash Screen)
